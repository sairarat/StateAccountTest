# Lab Seatwork 2 of State Account Test
State Account Test Program using State Design Pattern

# Requirements
* **Active Account**:
    Allows deposits and withdrawals.
    Can transition to **Suspended** or **Closed**.


* **Suspended Account**:
    Disallows deposits and withdrawals.
    Allows viewing account information.
    Can transition to **Active** or **Closed**.


* **Closed Account**:   
    Disallows all transactions and viewing of information. Terminal state: cannot be reactivated or suspended.
